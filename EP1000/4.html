Week 4: Chess Horse Piece

To do a chess horse piece like the one above, first you have to find a 2d image of a chess piece online. For this, I used the image below for my sketch.

Insert the image into Fusion 360. Start by tracing over the image using the sketch function. We want to keep it small so the specifications for my horse is 50mm tall, 30mm wide and has a thickness of 2mm.

Once you are done, extrude the sketch but make sure it is a symmetrical extrusion. I added details into the horse as you can see in the 3D files so we can see how the structures will hold up when we print it out. Before printing, you should check if the horse is hollow or not so we do not waste materials while printing.

Now, we export the file as an STL file and move on to Cura for setup

It would appear horizontal at first so make sure to rotate and make it upright. Then we do the settings. I did a 0.25mm print with a 5% infill because we do not need a lot of material for such a small figure. The base is a skirt so it does not stick to the printing base.

Why is this good for examining 3D printing?

Because there are overhangs and curves and that will create infills and supports. Not only that, the base is made from a revolution so we are utilising what we have learnt.
