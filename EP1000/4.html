Week 4: Chess Horse Piece

To do a chess horse piece like the one above, first you have to find a 2d image of a chess piece online. For this, I used the image on the right for my base.

Insert the image into Fusion 360. Start by tracing over the image using the sketch function. We want to keep it small so the specifications for my horse is 50mm tall, 30mm wide and has a thickness of 2mm.

Once you are done, extrude the sketch but make sure it is a symmetrical extrusion. I added details into the horse as you can see in the 3D files so we can see how the structures will hold up when we print it out

Why is this good for examining 3D printing?

Because there are overhangs and curves and that will create infills and supports.
