How all of these are done

For the first 4 models, there were measurements on the assignment page which I could follow. However, the fidget spinner did not have the measurements explicitly laid out, so I had to do some digging on the website to find the measurements for each component.

Ensure you start at the origin for every single model because there is a measurements next to the different axis. First, create a sketch and draw out the shape of the objects you are trying to create. In the case of the fidget spinner, since there were multiple parts, I started with "creating a component" first then starting the sketch.

In my case, I learn better by trying the random tools in the program until I see what works or not.

TIPS:The M8 in the M8 Stainless Steel Hex Nuts stands for mm. Always plan out the sketch before doing the model.
